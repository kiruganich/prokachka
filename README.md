## Задача 1 «PRIMARY SOURCES OR IT DIDN’T HAPPEN»
```
Скрипт автоматически извлекает пять фактов только из официальных первоисточников (NASA, IETF, Unicode, Bitcoin Core, OpenLibrary), парсит их и формирует флаг вида FLAG{...}. Есть unit-тесты, используется программный парсинг HTML/JSON/текста.

```
